Ecosystem Component Version

Summary

  • Ecosystem is application developement environment
  • Component is name of package/component
  • Dependencies indicates total number of dependencies component has

Version

  • Current indicates version of analysed component
  • Latest indicates with latest component's version
  • Recommended indicates latest version of component that is free of security and vulnerability

Code Metrics

  • Lines of code represents total number of lines in component
  • Cyclomatic complexity is a metric, used to indicate the complexity of a program It is a quantitative measure of the number of linearly independent paths through a program's source code
  • Number Of Files represents total number of files in component

CVE

  • Common Vulnerabilities and Exposures (CVE) provides a reference-method for publicly known information-security vulnerabilities and exposures
  • Common Vulnerability Scoring System (CVSS) is a free and open industry standard for assessing the severity of computer system security vulnerabilities
  • CVSS Scores range from 0 to 10, with 10 being the most severe.
